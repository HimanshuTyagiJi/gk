Example:
यदि एक सतह का तापमान अंतर 20°C है, तापीय चालकता 0.5 W/m°C है और क्षेत्रफल 1 m2 है, तो तापीय प्रवाह ज्ञात करें।
( Q = -0.5 W/m°C × 1 m2 × 20°C0.1 m = -100 W )

तरंग गति (Wave Motion)

तरंग सूत्र (Wave Formula)

( v = λ × f )

Example:
यदि तरंग की आवृत्ति 50 Hz है और तरंग की लंबाई 2 m है, तो तरंग की गति ज्ञात करें।
( v = 2 m × 50 Hz = 100 m/s )

ध्वनि तरंग (Sound Waves)

ध्वनि तरंग का सूत्र (Sound Wave Formula)

( v = √(B/ρ) )

Example:
यदि ध्वनि की तरंग गति 343 m/s है और वायुमंडल का घनत्व 1.2 kg/m3 है, तो वायुमंडल कीBulk Modulus ज्ञात करें।
( B = (343 m/s)2 × 1.2 kg/m3 = 1.401 × 105 N/m2 )

अप्रगामी तरंग (Standing Waves)

अप्रगामी तरंगों का सूत्र (Standing Waves Formula)

( λn = 2Ln )

Example:
यदि तार की लंबाई 1.5 m है और n1 की स्थिति पर तरंग की लंबाई ज्ञात करें।
( λ1 = 2 × 1.5 m1 = 3 m )

तनी डोरियों के कंपन (Vibrations of Stretched Strings)

डोरी के कंपन का सूत्र (Vibrations of Stretched Strings)

( f = 12L × √(T/μ) )

Example:
यदि डोरी की लंबाई 2 m है, तनाव 100 N है, और द्रव्यमान 0.01 kg है, तो प्रथम ओर्टन की आवृत्ति ज्ञात करें।
( f = 12 × 2 m × √(100 N / 0.01 kg) = 5 × 102 Hz )

व्यतिकरण एवं यंग का प्रयोग (Interference and Young's Experiment)

यंग के प्रयोग का सूत्र (Young's Experiment Formula)

( Δx = λLd )

Example:
यदि यंग के प्रयोग में द्वारों के बीच की दूरी 0.5 mm है, और लम्बाई 2 m है, तो विवर्तन की चौड़ाई ज्ञात करें।
( Δx = λ × 2 m0.5 mm )

विस्पंद एवं डॉप्लर प्रभाव (Beats and Doppler Effect)

विस्पंद का सूत्र (Beats Formula)

( nbeats = |f1 - f2| )

Example:
यदि दो ध्वनि तरंगों की आवृत्तियाँ 256 Hz और 260 Hz हैं, तो विस्पंद की आवृत्ति ज्ञात करें।
( nbeats = |256 Hz - 260 Hz| = 4 Hz )

डॉप्लर प्रभाव का सूत्र (Doppler Effect Formula)

( fo = fs × v + vov - vs )

Example:
यदि ध्वनि की गति 340 m/s है, स्रोत की आवृत्ति 500 Hz है, और स्रोत और पर्यवेक्षक के बीच की गति 30 m/s है, तो पर्यवेक्षक के लिए ध्वनि की आवृत्ति ज्ञात करें।
( fo = 500 Hz × 340 m/s + 30 m/s340 m/s - 30 m/s = 538 Hz )

प्रकाश का परावर्तन (Reflection of Light)

परावर्तन का सूत्र (Reflection Formula)

( θi = θr )

Example:
यदि एक प्रकाश किरण का परावर्तन कोण 30° है, तो इसका परावर्तन कोण भी 30° होगा।
( θi = 30° = θr )

लेंस द्वारा प्रतिबिंब (Image Formation by Lenses)

लेंस सूत्र (Lens Formula)

( 1f = 1v + 1u )

Example:
यदि लेंस की फोकल लंबाई 10 cm है और वस्तु की दूरी 20 cm है, तो प्रतिबिंब की दूरी ज्ञात करें।
( 1v = 110 - 120 = 1/30 )
तो v = 30 cm।

स्नेल का नियम एवं प्रिज्म से अपवर्तन (Snell's Law and Refraction through Prism)

स्नेल का नियम (Snell's Law)

( sin θ1sin θ2 = n2/n1 )

Example:
यदि पहली माध्यमा में कोण 30° है और दूसरी माध्यमा में कोण 45° है, तो अपवर्तनांक ज्ञात करें।
( n2/n1 = sin 30°sin 45° = 0.707/0.5 = 1.414 )

विक्षेपण क्षमता, दूरदर्शी तथा सूक्ष्मदर्शी (Dispersive Power, Telescope and Microscope)

विक्षेपण क्षमता (Dispersive Power)

( ω = ΔθΔλ )

Example:
यदि किसी पदार्थ की विक्षेपण क्षमता 1.5 है और रंगों की तरंग दैर्ध्य की अंतर 100 nm है, तो विक्षेपण कोण ज्ञात करें।
( Δθ = 1.5 × 100 nm = 150 nm )

वैधुत क्षेत्र एवं विभव (Electric Field and Potential)

वैधुत क्षेत्र का सूत्र (Electric Field Formula)

( E = Fq )

Example:
यदि एक बिंदु पर बल 10 N है और आवेश 2 C है, तो वैधुत क्षेत्र ज्ञात करें।
( E = 10 N2 C = 5 N/C )

वैधुत विभव का सूत्र (Electric Potential Formula)

( V = Wq )

Example:
यदि एक बिंदु पर कार्य 20 J है और आवेश 4 C है, तो वैधुत विभव ज्ञात करें।
( V = 20 J4 C = 5 V )

गौस की प्रमेय (Gauss's Theorem)

गौस की प्रमेय का सूत्र (Gauss's Theorem Formula)

( ∮ E · dA = Qencε0 )

Example:
यदि एक बंद सतह के लिए कुल आवेश 10 μC है, तो प्रवाह ज्ञात करें।
( ∮ E · dA = 10 μC8.85 × 10-12 F/m = 1.13 × 109 V·m )

वैधुत धारिता (Electric Capacitance)

वैधुत धारिता का सूत्र (Capacitance Formula)

( C = QV )

Example:
यदि एक संधारित्र में 6 μC आवेश है और विभव 3 V है, तो धारिता ज्ञात करें।
( C = 6 μC3 V = 2 μF )

वैधुत चलन एवं सरल परिपथ (Electric Current and Simple Circuits)

ओहम का नियम (Ohm's Law)

( V = IR )

Example:
यदि प्रतिरोध 5 Ω है और प्रवाह 2 A है, तो विभव ज्ञात करें।
( V = 2 A × 5 Ω = 10 V )

गतिमान आवेश एवं चुंबकीय द्विध्रुव का आघूर्ण (Moving Charges and Magnetic Dipole Moment)

चुंबकीय द्विध्रुव का आघूर्ण (Magnetic Dipole Moment)

( μ = m × l2 )

Example:
यदि दण्ड चुंबक की लंबाई 0.1 m है और उसमें बल 0.05 N है, तो चुंबकीय द्विध्रुव का आघूर्ण ज्ञात करें।
( μ = 0.05 N × 0.1 m = 0.005 A·m² )

बल्बनुमा क्षेत्र (Toroidal Field)

बल्बनुमा क्षेत्र का सूत्र (Toroidal Field Formula)

( B = μ0NI2πr )

Example:
यदि बल्बनुमा क्षेत्र में N = 100, I = 2 A, और r = 0.05 m है, तो चुंबकीय क्षेत्र बल ज्ञात करें।
( B = 4π × 10-7 × 100 × 22π × 0.05 = 0.0008 T )

स्थायी चुंबक, दण्ड चुंबक तथा गैल्वेनोमीटर (Permanent Magnet, Bar Magnet and Galvanometer)

गैल्वेनोमीटर का सूत्र (Galvanometer Formula)

( G = θI )

Example:
यदि गैल्वेनोमीटर का कोण विचलन 5° है और धारा 0.1 A है, तो गैल्वेनोमीटर की संवेदनशीलता ज्ञात करें।
( G = 5°0.1 A = 50°/A )

द्रव्यों का चुंबकीय गुण एवं ट्रांसफार्मर (Magnetic Properties of Materials and Transformer)

ट्रांसफार्मर का सूत्र (Transformer Formula)

( Vp/Vs = Np/Ns )

Example:
यदि प्राथमिक वलय में 100 टर्न हैं और द्वितीयक वलय में 50 टर्न हैं, और प्राथमिक वोल्टेज 220 V है, तो द्वितीयक वोल्टेज ज्ञात करें।
( Vs = Vp × Ns/Np = 220 V × 50/100 = 110 V )

प्रत्यावर्ती धारा (Alternating Current)

प्रत्यावर्ती धारा का RMS मान (RMS Value of Alternating Current)

( IRMS = I0√2 )

Example:
यदि I0 = 10 A है, तो प्रत्यावर्ती धारा का RMS मान ज्ञात करें।
( IRMS = 10√2 ≈ 7.07 A )

वैधुत चुंबकीय तरंगे (Electromagnetic Waves)

वैधुत चुंबकीय तरंगों का वेग (Velocity of Electromagnetic Waves)

( c = 1√(μ0ε0) )

Example:
यदि μ0 = 4π × 10-7 H/m और ε0 = 8.854 × 10-12 F/m है, तो वैधुत चुंबकीय तरंगों का वेग ज्ञात करें।
( c = 1√(4π × 10-7 × 8.854 × 10-12) ≈ 3 × 108 m/s )

बोहर का परमाणु मॉडल (Bohr's Atomic Model)

बोहर के परमाणु मॉडल का सूत्र (Bohr's Atomic Model Formula)

( En = -13.6 Z²n² eV )

Example:
यदि Z = 1 और n = 2 है, तो ऊर्जा स्तर ज्ञात करें।
( E2 = -13.6 1²2² eV = -3.4 eV )

प्रकाश वैधुत प्रभाव, एक्स-रे किरणें तथा रेडियो एक्टिवता (Photoelectric Effect, X-rays and Radioactivity)

प्रकाश वैधुत प्रभाव का सूत्र (Photoelectric Effect Formula)

( KE = hν - φ )

Example:
यदि hν = 5 eV और φ = 2 eV है, तो KE ज्ञात करें।
( KE = 5 eV - 2 eV = 3 eV )

एक्स-रे किरणों का सूत्र (X-rays Formula)

( λ = hmv )

Example:
यदि m = 9.11 × 10-31 kg और v = 3 × 108 m/s है, तो λ ज्ञात करें।
( λ = 6.63 × 10-349.11 × 10-31 × 3 × 108 ≈ 2.43 × 10-12 m )

रेडियो एक्टिवता का सूत्र (Radioactivity Formula)

( N = N0 e-λt )

Example:
यदि N0 = 100 और λ = 0.693 s-1 है, तो t = 1 s के बाद N ज्ञात करें।
( N = 100 e-0.693 × 1 ≈ 50 )

अर्धचालक तथा ट्रांजिस्टर (Semiconductors and Transistors)

ट्रांजिस्टर का एम्पलीफिकेशन फैक्टर (Amplification Factor of Transistor)

( β = ICIB )

( β = 5 × 10-350 × 10-6 = 100 )

संचार व्यवस्था (Communication Systems)

संचार व्यवस्था का एंटेना फार्मूला (Antenna Formula of Communication Systems)

( L = λ2 )
